# atscv
ATS-safe, LuaLaTeX-only, CTAN-oriented framework for CVs and cover letters.

## Usage
`\documentclass[lang=en,mode=modern,colorscheme=blue,profile=general,variant=default]{atscv}`

Compile with `lualatex` only.

## Features
- Single-column ATS-safe flow
- Modes: ats, modern, executive, consulting, compact
- Color schemes: black, blue, navy, slate, green, teal, burgundy, purple, brown, orange
- CV and cover-letter macros
- Multilingual via babel
- Validation scripts in `tests/`
- PDF/UA tagging postponed as future roadmap

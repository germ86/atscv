# Examples

- `minimal.tex`: shortest CV skeleton.
- `fabio-bigtech-modern.tex`: Fabio profile in modern mode.
- `fabio-executive.tex`: executive emphasis spacing/style.
- `fabio-consulting.tex`: consulting-style concise structure.
- `fabio-ats.tex`: strict ATS-focused appearance.
- `color-schemes.tex`: demonstrates supported color schemes.
- `multilang-*.tex`: multilingual compilation examples.

See docs: `../docs/atscv-manual.md` and `../README.md`.

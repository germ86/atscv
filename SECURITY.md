# Security Policy

## Data and secrets
- No secrets are required to build or test this repository.
- Do not commit personal/private generated CV data.

## Privacy reminder
CV and cover-letter text can contain personal data. Review output before sharing publicly.

## Reporting issues
Please report security or privacy concerns through GitHub Issues (or private channel if required by your organization), including:
- affected file(s)
- reproducible steps
- expected vs actual behavior
